# PRD V2 — ENTERPRISE VALUATION ENGINE

## Objetivo

Criar um webapp premium de valuation multicritério para:

- EAD
- SaaS
- Marketplaces
- Negócios digitais

## Funcionalidades

- DCF avançado
- EBITDA múltiplo setorial
- ARR/MRR
- Score inteligente
- Benchmark setorial
- Simulador de cenários
- Gráficos interativos
- Exportação PDF

## Stack

- Node.js
- Vue 3
- Chart.js
- GSAP
- Tailwind CSS
- SQLite

## Design

- Dark mode
- Glassmorphism
- Layout premium responsivo

## Fórmula Central

Valuation =
(DCF × 0.4) +
(EBITDA × 0.3) +
(ARR × 0.2) +
(Score × 0.1)
